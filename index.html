HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user.
  • "h#" is used to indicate a header and its level
  • "ul" and ol represent unordered and ordered lists respectively
  • "li" indicates list elements
  • "section" divides the body code into different sections
  • "img" is used to indicate an image element linked to its source
  • "span" and "div" indicate dividers on the webpage
